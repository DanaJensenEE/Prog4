Program Information

This program determine the amount of time a printer has been idle. Documents that need to be printed are placed into a queue in order of arrival time. A counter will record the idle time of the printer and return the time between printing documents, time to print each page, number of documents printed, the total idle time, and the number of documents still left to print if any. The program can get its documents from either randomly generated values or random values from a file.
